Testado en la version 2.7 de Python

Para ejecutar:

	- python "PATH Para el archivo"+ nombre del archivo ("proyectfinal.py") - por ejemplo: C:\Python27\python proyectfinal.py
	- como alternativa ejecutar el REPL de python y ejecutar en seguida el seguinte commando:
			execfile('PATH para el archive + nombre del archivo') 
	- Para cambiar el dataset:
		cambiar el codigo del PATH, el la linea que hace referencia a lectura del archivo:
		#Frecuent_ItemSet_DataSet = array(np.genfromtxt('.\..\data\grouptest.dat'), dtype='int')
	- Se puede cambiar el Min_Threshold directamente
